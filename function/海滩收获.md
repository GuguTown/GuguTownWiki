# 海滩收获
### 沙滩
沙滩上每个出现的物品都只会保留24小时,沙滩刷新后每1320分钟可再次刷新一次。
### 沙滩强制刷新
点击【强制刷新，立即获得下一批随机装备】，可以使用200000贝壳强制刷新沙滩，每日最多2次。
### 点击沙滩装备
点击沙滩上的装备图标，可以选择放入背包或是熔炼为护符（熔炼需要至少是稀有级别）
### 批量清理沙滩
点击【批量清理沙滩（清除史诗以下装备）】，可以将史诗以下装备全部转换为贝壳+强化护符所需的果核
